# 등산 트래커 (Hiking Tracker)

한국의 산악 정보를 바탕으로 위치를 기록하고 정상 정복을 저장하는 오프라인 기반 등산 트래커 어플리케이션입니다.

## 주요 기능
- **운동 시작**: 현재 위치부터 지도에 경로를 표시하고 시간, 거리, 평균 속도, 고도를 측정합니다.
- **사진 촬영**: 트래커 화면에서 코멘트와 함께 위경도를 포함한 사진을 남길 수 있습니다.
- **산 정상 정복**: 한국 주요 산의 정상 50m 반경에 도달 시 자동으로 축하 알림과 기록을 저장합니다.
- **운동 기록**: 저장된 이전 등산 기록들을 지도의 경로와 사진 핀을 통해 확인할 수 있습니다.
- **오프라인 동작**: 서버 통신 없이 내부 기기의 SQLite를 통해 모든 운동 데이터와 사진 정보를 안전하게 저장합니다.
