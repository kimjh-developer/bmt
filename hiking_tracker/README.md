# 등산 트래커 (Hiking Tracker)

한국의 산악 정보를 바탕으로 위치를 기록하고 정상 정복을 저장하는 오프라인 기반 등산 트래커 어플리케이션입니다.

## 주요 기능
- **운동 시작**: 현재 위치부터 지도에 경로를 표시하고 시간, 거리, 평균 속도, 고도를 측정합니다.
- **사진 촬영**: 트래커 화면에서 코멘트와 함께 위경도를 포함한 사진을 남길 수 있습니다.
- **산 정상 정복**: 한국 주요 산의 정상 50m 반경에 도달 시 자동으로 축하 알림과 기록을 저장합니다.
- **운동 기록**: 저장된 이전 등산 기록들을 지도의 경로와 사진 핀을 통해 확인할 수 있습니다.
- **오프라인 동작**: 서버 통신 없이 내부 기기의 SQLite를 통해 모든 운동 데이터와 사진 정보를 안전하게 저장합니다.

## 업데이트 기록 (Changelog)

### v1.0.3+1
- **UI/UX 전면 개편 (Stitch 디자인 적용)**
  - 밝은 테마에서 고급스러운 **다크 테마 (Tactical UI)**로 전면 교체
  - 시스템 폰트를 Noto Sans KR 및 Space Grotesk(영문/숫자)로 변경하여 가독성 강화
  - 컨테이너에 Glassmorphism(투명도 블러 렌더링) 효과 및 둥근 모서리 디자인 적용
- **주요 페이지 개선**
  - **TrackerPage**: 하단에 정밀 모드 / 배터리 절약 모드 토글 스위치 복원 및 UI 개선
  - **WorkoutSummaryPage**: 기존의 운동 종료 팝업을 전면 페이지 방식의 요약 화면으로 개편, 하단에 촬영한 사진 갤러리 추가
  - **RecordsPage**: 각 운동 기록을 카드형 세로 리스트 형태로 변경 및 썸네일 이미지 기능 적용
  - **RecordDetailPage**: 사진 클릭 시 전체 화면에서 고도, 사진 촬영 일시, 위치, 메모 정보를 직관적으로 띄우는 반투명 정보 오버레이 뷰어 기능 탑재
  - **MyPage**: '이번 달 기록' 통계 (누적 거리 및 상승 고도)를 자동 합산하도록 로직 업그레이드 및 1px 구분선 제거
